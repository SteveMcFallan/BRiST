# BRiST

> **Bridge without the tricky bits**

BRiST is a modern card game inspired by Bridge, designed to retain the strategy, partnership play and satisfaction of trick-taking while removing much of the complexity that makes traditional Bridge difficult for new players.

This repository contains the official implementation of **BRiST 2.0**, a Progressive Web App (PWA) that can be played in any modern browser on desktop, tablet or mobile.

## BRiST Rules

BRiST is played using a standard 52-card deck.

The game follows many familiar Bridge concepts while greatly simplifying bidding and scoring.

### Four-player game

- North partners South
- East partners West
- Dealer is also the Declarer
- Dealer's partner becomes Dummy
- Declarer controls both their own hand and Dummy

### Session play

Unlike Bridge rubber scoring, BRiST is played over a fixed number of hands.

Typical session lengths are:

- 8 hands
- 12 hands
- 16 hands (recommended)
- 20 hands

The partnership with the highest cumulative score wins the session.

## Installing

Clone the repository

```bash
git clone https://github.com/<username>/brist.git
```

Run locally

```text
Open index.html
```

or host using GitHub Pages.

Once published, install on your phone using

**Android**

Chrome → Install App

**iPhone**

Safari → Share → Add to Home Screen
